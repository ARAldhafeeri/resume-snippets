# resume-snippets
Since job boards that request you to re-enter resume information aren't going away, use resume-snippets to have quick copyable elements from your resumes.

## use chatgpt to quickly transform your resume into resume snippets settings
along with features like : filtering, searching, manging your snippets, resume snippets comes with a import/export settings feature, use the following prompt in ChatGPT to quickly transform your resume into settings :

```
transform my resume """paste your resume""" in this format """ [{"category":"Skills","content":"sdf","title":"sdf"},{"category":"Education","content":"sd","title":"sd"},{"category":"Projects","content":"sd","title":"test"},{"category":"Certifications","content":"sd","title":"sdf"},{"category":"SocialLink","content":"sd","title":"dd"},{"category":"PersonalInfo","content":"sd","title":"sdf"},{"category":"Work Experience","content":"sd","title":"sd"}]"""
```
